# DevOps Assignment

This project contains Terraform modules and a simple Node.js app used for a DevOps assignment.

## Quickstart

1. Configure AWS credentials (e.g. AWS CLI profile or environment variables).
2. From the project root run:
   - For deploy: `./scripts/deploy.sh`
   - For destroy: `./scripts/destroy.sh`

## Project structure

- `app/` - sample Node.js application
- `terraform/` - Terraform root module and child modules
- `scripts/` - helper scripts
